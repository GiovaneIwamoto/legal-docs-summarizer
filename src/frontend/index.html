Learn how to use
Select documents:
  1. You can select a folder from your computer through the attach file button. This option allows you to choose a folder in .zip format to extract the information;
  2. After choosing the .zip folder you want to open, you need to click the "Extract information" button;
  3. The information related to the selected folder will appear summarized below the selection and attachment fields;

Directory structure of the folder to be sent:
  1. The tool analyzes PDF files that are inside subfolders of the main folder;
  2. Therefore, the directory structure should be: 
                    
                  main-folder.zip/
                   |── subfolder1/
                  │    |── file1.pdf
                  │    |── file2.pdf
                   |── subfolder2/
                  │    |── file3.pdf
                  │    |── file4.pdf

Legal Document Summaries

This Generative AI tool integrates OCR for text extraction from PDF documents and uses AWS Bedrock to perform summarization. It uses Prompt Engineering to guide the generation of summaries that address critical aspects of legal content.
